# 🚀 Grafterr Landing Page (React)

A pixel-perfect, responsive landing page built using React based on the provided Figma design.

## 📌 Tech Stack
- React (Vite)
- JavaScript (ES6+)
- CSS3 (Custom styling, no frameworks)
- Custom Hooks

## ✨ Features
### Hero Section
- Gradient headline text
- Subheadline and CTA button
- Floating decorative shapes

### Features Section
- Section heading with accent text
- Responsive product carousel
- 3 product cards:
  - Point of Sale
  - Self-service
  - Kitchen Management

### Carousel Functionality
- Previous / Next navigation
- Smooth sliding animation (300ms)
- Responsive:
  - Desktop → 3 items
  - Tablet → 2 items
  - Mobile → 1 item

### Data Handling
- Data loaded dynamically from local JSON
- Simulated API delay (1–1.5 seconds)

### UI & UX
- Fully responsive design
- Gradient UI elements
- Hover effects and transitions
- Loading skeletons
- Error handling with retry option

## ⚙️ Setup Instructions
1. Clone the repository
2. Run: npm install
3. Run: npm run dev
